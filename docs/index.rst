Nichess
#######

Nichess is chess where pieces have abilities and health points.

Code: https://github.com/nichess-game

Hosted at: http://www.nichess.org

**Important:** Nichess is currently hosted with Heroku free tier, which means that it will shut down if no one is using it. In that case, you just need to wait 10 seconds and refresh the page.
